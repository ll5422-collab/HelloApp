
# HelloApp

HelloApp is a simple Java project that demonstrates basic programming concepts.

The project starts with displaying "Hello World" and gradually progresses to:
- Displaying a user name
- Accepting input from command line
- Managing names in a list
- Refactoring code into methods and classes
- Storing data across runs
- Displaying names in banner format

## How to Run

1. Compile the program:
    ```bash
    javac src/HelloApp.java
    ```

2. Run the program:
    ```bash
    java -cp src HelloApp
    ```
